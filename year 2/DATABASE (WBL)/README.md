## Course Overview

This course introduces database concepts, relational models, SQL, and database design for storing, managing, and retrieving structured data effectively.

---

## Reflection

Database helped me understand how data should be organised, stored, and managed in a structured way. This course showed me that databases are not only important for storing information, but also for ensuring that data can be accessed accurately, efficiently, and consistently.

Through this course, I learned about relational tables, keys, relationships, normalization, and SQL queries. These topics helped me see how real systems manage large amounts of data and how proper database design supports system quality. It also improved my ability to retrieve and manipulate data using SQL.

Overall, this course provided a strong foundation in data management. It is especially important for my studies because databases are a core component of data engineering, analytics, and many software systems.

---

## Key Takeaways

- Learned key database concepts and relational design.
- Improved SQL querying and data retrieval skills.
- Understood the importance of proper data organisation.
- Built strong foundation for data engineering and system development.

---

## Conclusion

In conclusion, **Database (WBL)** has provided useful knowledge and skills that are important for my academic development and future career. The course helped me improve my understanding, strengthen my learning foundation, and become more prepared to apply these concepts in real-world computing and professional situations.
