## Course Overview

This course introduces object-oriented programming concepts such as classes, objects, encapsulation, inheritance, polymorphism, and modular software design.

---

## Reflection

Object Oriented Programming helped me move from basic coding into a more structured way of developing software. It showed me that large programs can be organised more effectively by using objects and classes to represent real-world concepts.

The course improved my understanding of important ideas such as encapsulation, inheritance, and polymorphism. These concepts made me realise how reusable and maintainable code can be achieved through better design. It also helped me write programs in a more systematic and professional way.

Overall, this course strengthened my software development skills and prepared me for building larger applications. It is a very important course because object-oriented thinking is widely used in modern programming and system development.

---

## Key Takeaways

- Learned the core principles of object-oriented programming.
- Improved code structure and modular design skills.
- Understood the value of reusable and maintainable code.
- Prepared for larger-scale software development projects.

---

## Conclusion

In conclusion, **Object Oriented Programming** has provided useful knowledge and skills that are important for my academic development and future career. The course helped me improve my understanding, strengthen my learning foundation, and become more prepared to apply these concepts in real-world computing and professional situations.
