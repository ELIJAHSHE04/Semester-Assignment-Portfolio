## Course Overview

This course focuses on how users interact with computer systems. It covers usability, user experience, interface design, prototyping, user needs, evaluation, and human-centred design principles.

---

## Reflection

Human Computer Interaction helped me understand that a good system is not only about technical functions, but also about how easily users can understand and use it. A system may have many features, but it will not be effective if the interface is confusing or difficult to use.

Through this course, I learned the importance of user needs, usability principles, prototyping, and evaluation. It encouraged me to think from the user’s perspective and consider factors such as clarity, consistency, accessibility, and user satisfaction when designing an interface.

Overall, this course improved my awareness of user-centred design. It will be useful for future software development projects because it reminds me to build systems that are not only functional, but also practical, friendly, and meaningful for users.

---

## Key Takeaways

- Understood the importance of usability and user experience.
- Learned to design systems based on user needs.
- Practised thinking from the user’s perspective.
- Realised that good design improves system acceptance and effectiveness.

---

## Conclusion

In conclusion, **Human Computer Interaction** has provided useful knowledge and skills that are important for my academic development and future career. The course helped me improve my understanding, strengthen my learning foundation, and become more prepared to apply these concepts in real-world computing and professional situations.
